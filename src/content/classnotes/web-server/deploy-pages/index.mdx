Install git

[Donwlos Git](https://git-scm.com/downloads)

```bash
# apt-get install git
```

Create site folder

```bash
$ mkdir site-name
```

Open folder

```bash
$ code site-name
```

> Go to File > Menu > Open folder

![](/imgs/deploy/open-folder.png)

Initialize Git Repository

```bash
$ git init
```

![](/imgs/deploy/git-initialize.png)

Create site

![](/imgs/deploy/create-site.png)

Git add

```bash
$ git add -A
```

![](/imgs/deploy/git-add.gif)

Git commit

```bash
$ git config user.email "example@email.com"
$ git config user.name "example"
```

```bash
$ git commit -m "primeira versão do site"
```

![](/imgs/deploy/git-commit.gif)

Git remote add

```bash
$ git remote add origin https://github.com/lucachaves/site-name.git
```

![](/imgs/deploy/git-remote-add.gif)

Git push

```bash
$ git push -u origin master
```

![](/imgs/deploy/git-push.gif)

> Enter login and password

[git-credential-store - Helper to store credentials on disk](https://git-scm.com/docs/git-credential-store):

```bash
$ git config credential.helper store
$ git push
```
