### Specificity

Specificity Value: **NNNN**

| Number    | Selector                                         |
| --------- | ------------------------------------------------ |
| Thousands | style attribute                                  |
| Hundreds  | ID selector                                      |
| Tens      | class selector, attribute selector, pseudo-class |
| Ones      | element selector, pseudo-element                 |
