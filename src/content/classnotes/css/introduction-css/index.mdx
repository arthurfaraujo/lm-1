## Declaração do CSS

### Inline stylesheet

Atributo Style:

```plaintext
            declaration
          ┌──────┴───────┐
<p style="font-size: 20px;">Lorem ipsum dolor</p>
          └───┬───┘  └─┬─┘
          property   value
```

<CodeHtmlCssPreview src="/codes/css/introduction-css/css-inline" />

### Internal stylesheet

Blocos de Estilo e Tag Style:

```plaintext
selector
   ┌┴┐     declaration
    p { ┌──────┴───────┐
        font-size: 20px;
    }   └───┬───┘  └─┬─┘
        property   value
```

### Unidades de Medida

- [`<lenght>`](https://developer.mozilla.org/en-US/docs/Web/CSS/lenght): px, em, rem, cm, mm, in, pt, pc, vh, vw, vmin, vmax, %
- [`<color>`](https://developer.mozilla.org/en-US/docs/Web/CSS/color): `<named-color>`, `<hex-color>`, rgb(), rgba(), hsl(), hsla()
- [`<angle>`](https://developer.mozilla.org/en-US/docs/Web/CSS/angle): deg, rad, grad, turn
- [`<time>`](https://developer.mozilla.org/en-US/docs/Web/CSS/time): s, ms
