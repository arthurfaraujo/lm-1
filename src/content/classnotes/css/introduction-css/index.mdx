## Declaração do CSS

### Inline stylesheet

Atributo Style:

```
            declaration
          ┌──────┴───────┐
<p style="font-size: 20px;">Lorem ipsum dolor</p>
          └───┬───┘  └─┬─┘
          property   value
```

<CodeHtmlCssPreview code="/codes/css/introduction-css/css-inline" codeCss={false} />

### Internal stylesheet

Blocos de Estilo e Tag Style:

```
 selector
   ┌┴┐     declaration
    p { ┌──────┴───────┐
        font-size: 20px;
    }   └───┬───┘  └─┬─┘
        property   value
```
