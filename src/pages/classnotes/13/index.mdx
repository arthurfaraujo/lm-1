## Stretch a Box

### Absolute inside Absolute, with Fixed Dimensions

[demo/static-static-fixed-height.html](/codes/position/demo/static-static-fixed-height.html):

```html
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  position: static;
  width: 200px;
  border: 3px solid #000;
}

.child {
  position: static;
  width: 100px;
  height: 100px;
  background-color: blue;
}
```

<HyperText src="/codes/position/demo/static-static-fixed-height.html" />

### Absolute Inside Relative, with Fixed Dimensions
