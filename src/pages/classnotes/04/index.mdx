- [Facebook](https://www.flaticon.com/free-icon/facebook_145802#term=facebook&page=1&position=4)
- [Instagram](https://www.flaticon.com/free-icon/instagram_145805#term=instagram&page=1&position=7)
- [Twitter](https://www.flaticon.com/free-icon/twitter_179342#term=twitter&page=1&position=11)

[flat-icon/](flat-icon/):

```text
site
├── css
│   └── master.css
├── img
│   ├── facebook.svg
│   ├── instagram.svg
│   └── twitter.svg
└── index.html
```

```css
ul {
  list-style-type: none;
}

li {
  display: inline;
}

li img {
  width: 50px;
}
```

```html

  <li>
```
